# Portable Any Map для paint.net

Добавляет поддержку для `.pnm` файлов в графический редактор paint.net

## Форматы
* `P5` - загрузка, сохранение
* `P6` - загрузка, сохранение

## Примечания
При сохранении в чёрно-белый форматы нужно указать *один канал*, который 
будет сохранён, остальные будут проигнорированы. Если вы хотите сохранить 
цветное изображение как серое, сначале преобразуйте его встроенными 
инструментами редактора (меню "Коррекция" -> "Сделать чёрно-белым").

## Установка
1. Скачать плагин, например из Releases
2. Единственный `.dll` файл переместить в папку `FileTypes` там, куда установлен
   paint.net, например `C:\Program Files\paint.net\FileTypes`
